RuleGo provides Go APIs for visual configuration, which you can use to build custom visual editors or low-code platforms. These APIs handle component form definitions, rule chain configuration read/write, and layout information management.

If you need a ready-to-use visual editor, see [RuleGo-Server Visual Editor](/en/pages/rulego-server-editor/).

## API List

| API | Description |
|-----|-------------|
| [Get Component Configuration Forms](/en/pages/get-component-form/) | Scan registered components and generate form definitions |
| [Component Form Conventions](/en/pages/component-form-conventions/) | Config struct tags and auto-generation rules |
| [Get Rule Chain Configuration](/en/pages/get-rule-chain-config/) | Get rule chain DSL for display |
| [Get Rule Chain Node Configuration](/en/pages/get-rule-node-config/) | Get single node configuration |
| [Batch Initialize Rule Chains](/en/pages/batch-init-rule-chains/) | Batch load rule chains from a folder |
| [Delete Rule Chain Instance](/en/pages/delete-rule-chain-instance/) | Delete rule chain and release resources |
| [Save Rule Chain Layout](/en/pages/save-rule-chain-layout/) | Node layout coordinate storage |
