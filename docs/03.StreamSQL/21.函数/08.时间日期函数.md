# StreamSQL 时间日期函数

时间日期函数用于处理时间和日期数据。

## NOW - 当前时间函数
**语法**: `now()`  
**描述**: 返回当前的日期和时间。  
 
## CURRENT_TIME - 当前时间函数
**语法**: `current_time()`  
**描述**: 返回当前时间。  
 
## CURRENT_DATE - 当前日期函数
**语法**: `current_date()`  
**描述**: 返回当前日期。
