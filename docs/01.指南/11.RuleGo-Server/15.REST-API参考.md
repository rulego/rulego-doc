RuleGo-Server 所有 API 基于 `/api/v1` 路径，需要 JWT 或 API Key 认证（`require_auth = true` 时）。

在线 API 文档：[Apifox](https://apifox.com/apidoc/shared-d17a63fe-2201-4e37-89fb-f2e8c1cbaf40/234016936e0)

## 通用说明

### 认证

```http
Authorization: Bearer {jwtToken或apiKey}
```

或

```http
X-API-Key: {apiKey}
```

### 分页参数

| 参数 | 说明 |
|------|------|
| `page` | 页码，从 1 开始 |
| `size` | 每页数量 |

### 权限标识

格式为 `resource:action`，如 `rule:read`、`rule:write`。

## 规则链

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/api/v1/rules` | `rule:read` | 列出规则链 |
| GET | `/api/v1/rules/:id` | `rule:read` | 获取规则链 DSL |
| POST | `/api/v1/rules/:id` | `rule:write` | 创建/更新规则链 |
| DELETE | `/api/v1/rules/:id` | `rule:delete` | 删除规则链 |
| GET | `/api/v1/rules/:id/latest` | `rule:read` | 获取最新规则链（`:id` 被忽略） |
| POST | `/api/v1/rules/:id/base` | `rule:write` | 更新规则链基础信息 |
| POST | `/api/v1/rules/:id/config/:varType` | `rule:write` | 保存规则链配置变量 |
| POST | `/api/v1/rules/:id/operate/:type` | `rule:operate` | 部署操作（start/stop/set-to-main） |
| POST | `/api/v1/rules/:id/notify/:msgType` | `rule:execute` | 异步执行规则链 |
| POST | `/api/v1/rules/:id/execute/:msgType` | `rule:execute` | 同步执行规则链 |
| POST | `/api/v1/rules/:id/v1/chat/completions` | `rule:execute` | OpenAI 兼容对话端点 |
