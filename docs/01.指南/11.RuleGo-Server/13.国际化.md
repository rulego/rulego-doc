RuleGo-Server 支持多语言国际化，提供语言包的管理 API。

## 列出语言包

```http
GET /api/v1/locales
Authorization: Bearer {token}
```

返回所有已注册的语言包列表。

## 获取指定语言包

```http
GET /api/v1/locales?lang=zh-CN
Authorization: Bearer {token}
```

## 保存/更新语言包

```http
POST /api/v1/locales
Authorization: Bearer {token}
Content-Type: application/json

{
  "lang": "zh-CN",
  "data": {
    "key": "值",
    "node.filter": "过滤器"
  }
}
```

语言包数据存储在 `data/locales/` 目录下。

## 在编辑器中使用

编辑器支持通过 API 或配置设置语言：

```javascript
editor.setLocales({ "zh-CN": { ... } })
```

组件名称、分类名称、表单标签等都会根据语言包进行本地化显示。

## 权限

| 操作 | 所需权限 |
|------|----------|
| 查看语言包 | `locale:read` |
| 修改语言包 | `locale:write` |
