RuleGo-Server 内置 100+ 组件，同时支持通过共享节点池和动态组件机制进行扩展。

## 组件目录

获取所有已注册组件及其表单定义：

```http
GET /api/v1/components
Authorization: Bearer {token}
```

返回内容包括：
- **endpoints**：输入端点组件（HTTP、MQTT、Kafka、Cron、MCP Server 等）
- **nodes**：处理节点组件（过滤器、转换器、动作节点等）
- **tools**：工具组件（AI 工具等）
- **builtins**：内置处理器（含 `globals` 全局变量名列表）

### 组件分类

| 分类 | 说明 | 示例组件 |
|------|------|----------|
| `filter` | 过滤/路由 | `jsFilter`、`luaFilter` |
| `transform` | 数据转换 | `jsTransform`、`jsonTransform` |
| `action` | 执行动作 | `restApiCall`、`dbClient`、`mqttClient` |
| `external` | 外部集成 | `kafkaProducer`、`redisClient` |
| `flow` | 流程控制 | `fork`、`join`、`delay` |
| `endpoint` | 输入端点 | `endpoint/rest`、`endpoint/mqtt`、`endpoint/mqttServer` |
| `ai` | AI 相关 | `ai/agent` |

## 共享节点

共享节点允许在多个规则链中复用同一组配置。例如数据库连接、MQTT 客户端等。

### 列出共享节点

```http
GET /api/v1/shared-nodes?page=1&size=20
Authorization: Bearer {token}
```

### 创建/更新共享节点

```http
POST /api/v1/shared-nodes/{id}/{type}
Authorization: Bearer {token}
Content-Type: application/json

{
  "name": "MySQL连接",
  "configuration": {
    "driverName": "mysql",
    "dsn": "root:root@tcp(127.0.0.1:3306)/test",
    "poolSize": 10
  }
}
```

- `{id}`：节点 ID
- `{type}`：组件类型，如 `dbClient`、`mqttClient`

### 获取共享节点

```http
GET /api/v1/shared-nodes/{id}/{type}
Authorization: Bearer {token}
```

### 删除共享节点

```http
DELETE /api/v1/shared-nodes/{id}/{type}
Authorization: Bearer {token}
```

### 在规则链中引用共享节点

在节点配置中使用 `ref://` 前缀引用共享节点：

```json
{
  "type": "dbClient",
  "name": "查询数据库",
  "configuration": {
    "driverName": "ref://mysqlConn"
  }
}
```

## 动态组件

动态组件支持运行时安装和卸载，无需重启服务。

### 列出动态组件

```http
GET /api/v1/dynamic-components?page=1&size=20
Authorization: Bearer {token}
```

### 安装/升级组件

```http
POST /api/v1/dynamic-components/{id}
Authorization: Bearer {token}
Content-Type: application/json

{组件 DSL JSON}
```

### 获取组件定义

```http
GET /api/v1/dynamic-components/{id}
Authorization: Bearer {token}
```

### 卸载组件

```http
DELETE /api/v1/dynamic-components/{id}
Authorization: Bearer {token}
```

## 组件表单

每个组件的配置表单由后端自动生成，用于可视化编辑器渲染属性面板。

### 字段类型

| 类型 | 说明 |
|------|------|
| `string` | 文本输入 |
| `bool` | 布尔开关 |
| `int` | 整数输入 |
| `array` | 数组 |
| `map` | 键值对 |
| `struct` | 嵌套结构体 |

### 组件类型（UI 渲染）

| 组件 | 说明 |
|------|------|
| `codeEditor` | 代码编辑器（CodeMirror） |
| `textarea` | 多行文本 |
| `select` | 下拉选择 |

详细约定参见 [组件配置表单约定](/pages/component-form-conventions/)。
