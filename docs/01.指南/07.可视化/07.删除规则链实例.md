从规则引擎池中删除指定规则链实例，释放其占用的资源。

## API

```go
rulego.Del(chainId string) bool
```

- **参数**：`chainId` — 规则链 ID
- **返回值**：`true` 表示删除成功，`false` 表示规则链不存在

## 示例

```go
// 删除指定规则链
rulego.Del("rule01")
```

删除操作会：
1. 从引擎池中移除该规则链
2. 调用规则链中每个节点的 `Destroy()` 方法释放资源
3. 解除规则链内节点之间的关联

> 删除后无法恢复，如需重新使用需重新加载。
