RuleGo 提供用于可视化配置的 Go API，你可以基于这些 API 开发自定义的可视化编辑器或低代码平台。这些 API 负责组件表单定义、规则链配置的读写和布局信息管理。

如果你需要开箱即用的可视化编辑器，参见 [RuleGo-Server 可视化编辑器](/pages/rulego-server-editor/)。

## API 列表

| API | 说明 |
|-----|------|
| [获取组件配置表单](/pages/get-component-form/) | 扫描注册组件，生成表单定义 |
| [组件配置表单约定](/pages/component-form-conventions/) | Config 结构体 tag 和自动生成规则 |
| [获取规则链配置](/pages/get-rule-chain-config/) | 获取规则链 DSL 用于回显 |
| [获取规则链节点配置](/pages/get-rule-node-config/) | 获取单个节点配置 |
| [批量初始化规则链](/pages/batch-init-rule-chains/) | 从文件夹批量加载规则链 |
| [删除规则链实例](/pages/delete-rule-chain-instance/) | 删除规则链并释放资源 |
| [保存规则链坐标信息](/pages/save-rule-chain-layout/) | 节点布局坐标的存储方式 |
