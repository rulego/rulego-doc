创建通知（OnCreated Advice）：在规则引擎创建完成之后执行。

## 接口

```go
// Order 返回执行顺序，值越小，优先级越高
Order() int
// 规则链初始化时候会调用该方法创建新的实例，如果字段值需要继承，在这里处理
New() Aspect
// OnCreated 规则引擎成功创建之后的增强点
OnCreated(chainCtx NodeCtx)
```
