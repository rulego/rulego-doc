Debug Aspect：节点调试切面，是系统内置的切面，如果注册了`Config.OnDebug`，会自动启动该切面。 用于跟踪节点执行情况、出参、入参、耗时等信息。
详情参看[OnDebug](/pages/config/#ondebug) 。

:::tip
该切面会自动引入，不需要手动引入。
:::
